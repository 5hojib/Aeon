* `TELEGRAM_API` 
* `TELEGRAM_HASH`
* `DATABASE_URL` 
* `HEROKU_EMAIL` 
* `HEROKU_API_KEY` 
* `HEROKU_APP_NAME`

2. Fill other variable in config.env
3. Upload token.pickle and other private files from bsetting.
